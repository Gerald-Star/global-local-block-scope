Global Scope- Local Scope and Block Scope

Global Scope

The scope of a variable is its visibility; where in the code is the function available? Here is a list of the different scopes a variable can have. Global scope: The variable is available throughout the code

task is to understand how adding var (and not adding) before a variable name, can change the variable’s scope. When you add var before the variable name, its scope is determined by where it is placed. Like so:

Local Scope

Local scope: Available in only a certain area (like only within function)

Block Scope

Block scope: Available within an even more certain area (like an if-statement)

Example

When you add var before the variable name, its scope is determined by where it is placed. Like so:


var num1 = 18; // Global scope
function fun() {
var num2 = 20; // Local (Function) Scope
if (true) {
var num3 = 22; // Block Scope (within an if-statement) } }

Variable declared without any var or scope. It all becomes global scope.

Local Scope and Functions

Variables which are declared within a function, as well as the function parameters, have local scope. That means they are only visible within that function.
